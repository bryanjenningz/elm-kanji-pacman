# Elm Kanji Pacman

Learn Kanji by playing Pac-Man!

Built with [Elm](https://elm-lang.org) and [Elm Land](https://elm.land).

## Screenshots

<img width="350" src="./screenshot.png" alt="">

## Local development

```bash
# Requires Node.js v18+ (https://nodejs.org)
npx elm-land server
```
